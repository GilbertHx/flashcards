# Running the project
_____________________


Install all dependencies: `yarn install`

To start: `yarn start`


# Viewing the app on your phone:
* Install [Expo](https://expo.io/)
* Scan the QR-Code

# Features:
* Create decks
* Add questions
* Take a quiz
